# Student Management System

## Introduction

Student Management System is a console-based application developed using Object-Oriented Programming (OOP) principles in Python. The system helps manage students, lecturers, classes, and related information through a menu-driven interface.

## Technologies Used

* Python
* Object-Oriented Programming (OOP)
* JSON
* TXT File Handling

## Features

### Student Management

* Add Student
* Update Student Information
* Delete Student
* View Student List

### Lecturer Management

* Add Lecturer
* Update Lecturer Information
* Delete Lecturer
* View Lecturer List

### Class Management

* Create Classes
* Edit Class Information
* Add Students to Classes
* Remove Students from Classes
* Add Lecturers to Classes

### Data Processing

* Export Student List to TXT File
* Load Data from JSON File
* Save Data to JSON File

